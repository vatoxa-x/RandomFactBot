# Бот для озвучивания случайных фактов
##  Доступные команды
```
/start
```
Привецтвует пользователя
```
/fact
```
отправляет случайный факт

## Как использовать код
```
py -m venv venv
```
```
pip install pyttsx3
```
```
pip install pyTelegramBotAPI
```
### В файл config.py добавляем токен из официального бота BotFather
