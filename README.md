  # RandomFactBot

  🤖 *Умный Telegram-бот, который каждый день делится с пользователями случайными и удивительными фактами обо всем на свете.*

---

### ✨ Ключевые возможности (Features)
* 🎲 **Случайные факты** — генерация уникального факта в один клик.
* ⏰ **Ежедневная рассылка** — автоматическая отправка факта дня подписчикам.
* 📚 **Категории** — выбор интересующих тем (наука, история, космос).
* 🌐 **Мультиязычность** — поддержка русского и английского языков.

---

### ⚙️ Стек технологий (Tech Stack)
* **Язык**: Python
* **Библиотека**: `aiogram` (или другая библиотека, которую вы используете)
* **Логика**: Кастомный движок в `logic.py`

---

### 🚀 Быстрый старт (Installation)

#### 1. Клонирование репозитория
```bash
git clone https://github.com/vatoxa-x/RandomFactBot.git
cd RandomFactBot
```

#### 2. Настройка конфигурации
Заполните ваши токены и ключи в файле `config.py`:
```python
# config.py
BOT_TOKEN = "ВАШ_ТЕЛЕГРАМ_ТОКЕН"
```

#### 3. Запуск бота
```bash
python bot.py
```

---

### 📂 Структура проекта
* `bot.py` — точка входа и инициализация обработчиков бота.
* `config.py` — конфигурационный файл для хранения токенов и настроек.
* `logic.py` — основная бизнес-логика и парсинг фактов.
